Inserindo Imagens em HTML5

1. Tag <img>

A tag <img> é usada para incorporar imagens em páginas HTML.

Ela é uma tag vazia (não possui fechamento) e precisa, no mínimo, dos atributos src e alt.

2. Atributos essenciais

  • src: Caminho da imagem. Pode ser relativo ou absoluto.
  • alt: Texto alternativo para acessibilidade e SEO.

3. Dimensionamento

Você pode usar os atributos width e height para definir o tamanho da imagem.

Exemplo:

<img src="foto.jpg" alt="Pessoa sorrindo" width="300">

4. Boas práticas

  • Sempre use o atributo alt.
  • Organize suas imagens em uma pasta como img/ ou images/.
  • Otimize as imagens para carregamento mais rápido.
  • Prefira caminhos relativos quando possível.

5. Exemplo completo


<img src="imagens/logo-empresa.png" alt="Logotipo da empresa" width="150">
<img src="https://picsum.photos/400/250" alt="Paisagem aleatória" width="400" height="250">
<img src="imagens/icone-info.png" alt="Ícone de informação" width="50">
